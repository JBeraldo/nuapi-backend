- Make
## Steps to run
 
 - Clone the project
    ```bash
    git clone https://github.com/JBeraldo/nuapi-backend.git
    cd nuapi-backend
    ```
 - Run with make
    ```bash
    cp .env.example .env
    make dev
    ```
